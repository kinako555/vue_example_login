# vue_example_login
vue vue_router vuexを使用したtoken認証でのログイン画面作成   
サーバーサイドはRailsのdevise_auth_tokenを使用しています。  
クライアントのtokenもそれにもとずくものとなっています。

## 現機能
ユーザー作成    
ログイン    
ログアウト
